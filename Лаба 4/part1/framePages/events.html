Модель событий. Объект event.

События (events) представляют собой извещения, которые генерируются в результате действий пользователя или изменения состояния документа или окна. Динамический HTML предоставляет мощную модель событий, которая тесно связана со структурой документа. Модель событий основана на двух важнейших элементах, служащих для управления поведением документа: «всплывание» событий; действия по умолчанию. Всплывание событий является элементом модели событий, обеспечивающим просмотр иерархической структуры документа при обработке извещений событий. В результате такого просмотра на любое событие может быть получена реакция от всех родительских элементов в иерархии контейнеров, а не только лишь от элемента, в котором возникло событие. Другими словами, каждое действие, происходящее в элементе, его родительском элементе и так далее вплоть до тела документа, и, в конце концов – самого документа, получает извещение события. Событие может быть обработано на любом уровне, что позволяет создавать компактные и вместе с тем гибко настраиваемые программы. Такая обработка стала возможной за счет введения в динамическом HTML механизма связывания событий, которое представляет собой создание связи между определенным событием и программой. Действия по умолчанию обеспечивают встроенные в браузеры стандартные действия по обработке события. Например, действием по умолчанию в ответ на нажатие ссылки является переход по указанному URL и загрузка документа. В объектной модели динамического HTML можно заменять эти стандартные действия на другую модель поведения. Такая замена действия по умолчанию уже использовалась в примере 4.2.5 - когда вместо перехода по ссылке в элементе Area происходил вызов функции, генерирующей сообщение, действие по умолчанию отменялось. Для управления механизмом связывания событий динамический HTML предоставляет доступ к информации о событии и управление посредством такого доступа всплыванием события и действием по умолчанию. Данная информация предоставляется объектом event, который является свойством объекта window.
