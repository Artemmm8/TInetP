Описание задания

Демонстрация использования всех Escape-последовательностей:


Возврат на одну позицию \b
alert("Предложение для\bпыток нашим кодом")

Табуляция \t
alert("Предложение для \tпыток нашим кодом")

Переход на следующую страницу \f
alert("Предложение для \fпыток нашим кодом")

Переход на следующую строку \n
alert("Предложение для \nпыток нашим кодом")

Возврат каретки \r
alert("Предложение для \rпыток нашим кодом")

Двойная кавычка \"
alert("Предложение для \"пыток\" нашим кодом")

Одинарная кавычка - апостроф \'
alert("Предложение для \'пыток\' нашим кодом")

Обратная косая черта - обратный слэдж \\
alert("Предложение для \\пыток\\ нашим кодом")

Примечание: в зависимости от браузера и операционной системы некоторые escape-последовательности могут не работать.
