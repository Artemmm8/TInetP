Демонстрация работы метода prompt и alert:


prompt("Как вас зовут?", "Петров Иван Петрович" (данные по умолчанию));

alert("Предупреждение!");

Prompt (запрос/ответ) – служит для вывода информации и позволяет пользователю ввести ответ с клавиатуры

Alert (предупреждение) – служит для вывода информации
