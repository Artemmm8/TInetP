Описание задания

Присваивание значений переменным на лету, а также повторное использование изменённых переменных:


fio = prompt("Как вас зовут? (введите свои данные вместо данных по умолчанию)", "Жуковский Д. С.");

alert("Вы ввели: " + fio)

Base_fio = fio

fio = prompt("Теперь введите фамилию по матери. (введите свои данные вместо данных по умолчанию)", "Язвинский Д. С.");

alert("Первая фамилия: " + Base_fio)

alert("Вторая фамилия: " + fio)
