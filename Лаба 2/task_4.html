Описание задания

Демонстрация использования префиксной и постфиксной арифметической операции:


var x1 = 5;

var y1 = 10;

var result1 = x1-- + y1

alert( result1 ); //выведет 15



var x2 = 5;

var y2 = 10;

var result2 = --x2 + y2;

alert( result2 ); //выведет 14



Примечание: постфиксная форма x++/x-- отличается от префиксной ++x/--x тем, что возвращает старое значение своего аргумента, бывшее до увеличения (уменьшения).
